Data Storage

Where User Data Is Saved

Understand how login, camera details, alerts and evidence are stored.

Local Database

User accounts, cameras and alert records are saved in school_surveillance.db inside the project folder.

Evidence Folder

Captured screenshots are saved in the evidence folder and linked to alert records.

Another Device Login

Another device can log in only when it opens the same running server using your laptop IP or when the app is deployed online.

Cloud Use

For real multi-device login anywhere, deploy the Flask app on a cloud server and use a cloud database.
